# lucca-ai-tips

a curated collection of practical ai tips and tricks.

## purpose

provide bite‑size, actionable advice for developers, creators, and anyone using ai tools.

## structure

- `tips/` – markdown files, each containing a single tip.
- `scripts/` – utilities for generating and formatting tips.
- `docs/` – optional longer guides.

## contribution
